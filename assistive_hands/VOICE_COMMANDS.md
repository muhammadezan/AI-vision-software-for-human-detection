# AssistiveHands Voice Commands

Use Chrome or Edge and allow microphone permission when the browser asks. Voice starts automatically when the app loads. If the browser blocks automatic microphone startup, click the `Voice` button once to start listening.

Voice only types text when you give an explicit typing command, such as `type hello`. If a phrase is not recognized as a command, it will not be typed.

Control commands run as soon as they are detected. Typing commands wait for the final recognized phrase to avoid accidental text.
